Тема: Услуги спортзала. Функционал: Данный код предоставляет небольшое меню, в котором можно выбрать взаимодействие с объектами. Тут объектами являются услуги, которые может предложить спортзал. Их можно создать самому, вывести информацию на экран, изменить уданные и удалить объект полностью. Так же добавлена небольшая функция, с помощью которой можно выбрать услугу, пользоваться ей и следить за её статусом.
