# Тема: Услуги спортзала. 
Функционал: Данный код предоставляет небольшое меню, в котором можно выбрать опцию взаимодействия с объектами. Тут объектами являются услуги, которые может предложить спортзал. Их можно создать самому, вывести информацию на экран, изменить уданные или удалить полностью. Так же добавлена небольшая функция, с помощью которой можно управлять вашими улсугами. Вы сможете выбрать услугу, пользоваться ей, следить за её статусом или отказаться от выбранной услуги.
