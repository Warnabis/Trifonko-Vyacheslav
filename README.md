# Тема: Услуги спортзала. 
В этом проекте будут представлены:
+ Реализация CRUD-операций для работы с услугами.
+ Возможность выбрать для себя тренировку из списка доступных.
+ Следить за статусом услуг и отметка использования услуги.
+ Выбрать другую услугу.
